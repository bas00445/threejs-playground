## Getting Started

### Install packages

```
pnpm install

or

yarn install

or

npm install
```

### Start Storybook

```
pnpm storybook

or

yarn storybook

or

npm run storybook
```

### Slide for presentation

open the file `threejs101.excalidraw` on [Excalidraw](https://excalidraw.com/)
